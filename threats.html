القرصنة الإلكترونية

القرصنة الإلكترونية هي محاولات للوصول غير المصرح به إلى أنظمة، بيانات أو شبكات بأهداف مختلفة — مالية، تجسسية، سياسية أو لمجرد التحدي. المعرفة بأنواع الهجمات وكيف تفكر الجهة المهاجمة يساعدك تحمي نفسك قبل ما يصير الضرر.

أشهر أنواع الهجمات

  1. حقن SQL (SQL Injection) — إدخال استعلامات ضارة للوصول أو تعديل قواعد البيانات.
  2. البرمجة النصية عبر المواقع (XSS) — حقن سكربتات تعمل في متصفح الضحية عبر محتوى غير معالج.
  3. تزوير الطلبات عبر المواقع (CSRF) — إجبار مستخدم مسجّل على تنفيذ إجراءات عبر روابط أو نماذج مخادعة.
  4. كسر المصادقة وإدارة الجلسات — سرقة جلسات أو كلمات مرور، ضعف في إدارة الجلسات.
  5. تعريض البيانات الحسّاسة — نقل/تخزين بيانات دون تشفير أو حماية مناسبة.
  6. مكوّنات تحتوي ثغرات معروفة — استخدام مكتبات/حزم قديمة يمكن استغلالها.
  7. تهيئة أمان خاطئة (Security Misconfiguration) — إعدادات سيرفر/خدمات غير آمنة أو افتراضية.
  8. حقن أوامر / تنفيذ أوامر عن بعد (Command Injection / RCE) — إدخال يسمح بتنفيذ أوامر نظام على الخادم.
  9. إلغاء التسلسل غير الآمن (Insecure Deserialization) — استقبال كائنات مسلسلة قابلة للتعديل تؤدي لسلوك ضار.
  10. حقن XML / XXE — استغلال محللات XML لتحميل موارد محلية أو خارجية ضارة.
  11. Server‑Side Request Forgery (SSRF) — إجبار الخادم على طلب موارد داخلية أو خارجية باسم المهاجم.
  12. التحكم بالوصول المعطّل (Broken Access Control) — وصول غير مصرّح به إلى بيانات أو وظائف بسبب تحقق وصول ضعيف.
